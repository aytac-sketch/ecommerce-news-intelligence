# E-Commerce News Intelligence MVP

RSS + web kaynak + manuel link yapısıyla e-ticaret, pazaryeri, martech, growth ve regülasyon haberlerini toplayan; salt haber akışı ve AI analiz katmanı üreten MVP.

## Ana modüller
- Haber Akışı: Kaynak, tarih, kategori, özet ve orijinal link
- Intelligence: Satıcı etkisi, ajans fırsatı, içerik fikri, risk/fırsat skoru
- Kaynak Yönetimi: RSS, web sayfası, manuel kaynaklar
- Manuel Link Ekleme: RSS olmayan haberleri sisteme alma

## Teknoloji önerisi
- Next.js App Router
- Supabase/PostgreSQL
- TypeScript
- rss-parser
- cheerio/readability ileriki aşama için
- OpenAI API veya seçilecek LLM
